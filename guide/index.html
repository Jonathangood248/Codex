Welcome to the Task Guide!

Choose a task from the sidebar to get started. Each task will teach you how to edit a different part of the Habit Tracker app.

You'll learn to change CSS, HTML, and JavaScript — the three core languages of the web.

Tips before you start:

  • Open the project folder in VS Code
  • Keep the browser open next to VS Code
  • After editing a file, save it and refresh the browser
  • Use the "Check" button to verify your changes
